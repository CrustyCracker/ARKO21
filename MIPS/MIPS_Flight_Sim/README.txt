#-------------------------------------------------------------------------------
# Name: CrustyCracker
# Ball Flight Simulator in 64FPS
# The Program draws a pixel in the balls position every 1/64 of a secound
# User is asked for initial height, horisontal valocity and the gravitational acceleration
# so the program can draw a graph for a ball with given variables
# The ball is losing 1/9 of its valocity when it bounces
#-------------------------------------------------------------------------------
# The Program only supports 24bit 1600x800px .bmp files
#-------------------------------------------------------------------------------

Map1.bmp:
pokazuje lot 4 czarnej, czerwonej, zielonej, niebieskiej rzconych z wysokości na Ziemi.
Piłki miały prędkość kolejno 200, 100, 50 i 25 m/s dla 
g = 9.81 m/s^2

Map2.bmp:
pokazuje lot żaroodpornej piłki wyrzuconej z wysokości 800 metrów od słońca z prędkością 123 m/s dla
c = 275 m/s^2

Map3.bmp:
Pusta mapa gotowa do wykorzystania przez użytkownika
